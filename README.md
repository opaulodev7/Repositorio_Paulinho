Portfólio profissional moderno desenvolvido para apresentar projetos, habilidades técnicas e trajetória acadêmica como estudante de Análise e Desenvolvimento de Sistemas (ADS) no SENAI.

O projeto foi construído com foco em:

- visual moderno;
- responsividade;
- performance;
- simplicidade de manutenção;
- experiência do usuário.
